# HRA Web Analytics Dashboard

**Team Crimson**

A static analytics dashboard built for Indiana University's CNS Center, analyzing web log data from the Human Reference Atlas (HRA) ecosystem and the CNS Center website. The dashboard surfaces how researchers around the world interact with HRA tools — including the Registration User Interface (RUI), Exploration User Interface (EUI), and Cell Distance Explorer (CDE) — and identifies opportunities to improve usability, discoverability, and performance.
